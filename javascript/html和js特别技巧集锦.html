  • 网页是否被检索

    其中属性值有以下一些:
    属性值为”all”: 文件将被检索，且页上链接可被查询；
    属性值为”none”: 文件不被检索，而且不查询页上的链接；
    属性值为”index”: 文件将被检索；
    属性值为”follow”: 查询页上的链接；
    属性值为”noindex”: 文件不检索，但可被查询链接；
    属性值为”nofollow”: 文件不被检索，但可查询页上的链接。
