## vue-cli单页面脚手架
```bash
1、cd到总文件夹
2、cnpm install -g vue-cli
3、vue init webpack ***  （webpack表示一种模式，***表示所需的项目名称）
       如果安装vue1.0  webpack#1.0
4、回车...（前三个yes，后三个no）
5、cd 到项目目录
6、cnpm install
7、cnpm run dev

8、执行cnpm run build  （启动编译，生成编译后的dist文件）
```
